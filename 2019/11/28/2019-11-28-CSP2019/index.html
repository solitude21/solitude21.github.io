CSP2019-S2 流水账式的总结与反思

发表于 2019-11-28 更新于 2020-04-04 分类于 essay

  或许多年后，回想高中时的OI生涯，想到能有一群志同道合的人一起奋斗，想起曾经的奋斗与努力，没能取得好成绩的挫败感，也早已经被时间淡去了。毕竟，我努力过，我没有遗憾。

Day0

  从下午2点开始和Qxx，Lj，Shx在机房打模板（事实证明并没有什么用），事实上，这确实是我最后一次打模板，因为我考试时一个都没有用上……

  最后一次机房集训了。到晚上6点，大家都还不舍得离开。在这种时候，我心中五味杂陈。既想早点结束这一切逃离这里，又希望能多留一会和同伴度过考前最后一天。真是矛盾。

  晚上我还在打模板（徒劳）。现在看来，我应该在考前的晚上好好休息，调整好心态。

Day1

  7点25分左右起床，然后就是上洛谷，发了一句“CSP-S 2019 RP++”迷信一波。考场就在学校（传统艺能），所以我和平常上学一样骑车，10分钟就到了。

  考试前，和同学聊聊天，上卫生间，缓解一下紧张的心情。

  考试的地点是7楼的语音室。桌子上有一块凸起，键盘放着很不方便。而且那种键盘我好久没有用过，很不适应。公布解压密码后，我输入了好几遍都没对。后来才发现我应该输入数字“0”而不是字母“O”……还是太慌张了。

  前二十分钟读题，确定了三题都打暴力的基本方针……

  T1是个找规律题，先打了个暴力，再打正解对拍。用了30分钟左右。此时我突然发现我脖子右侧有些酸痛，虽说不像落枕时那么痛，但也挺影响发挥的。

  T2就没那么顺利了。考前除了字符串哈希，我就没看过和处理字符串相关的内容。一开始想了个只能拿20分的O（n^3）算法，先打上去再说，结果调了好久。码力还是太差！打完后已经10点多了，我打算先看T3。

  看到T3，先打10分暴力。又调了一会，搞到11点才弄完。打完暴力，我发现“菊花图”的情况有25分，用草稿纸算了一会，感觉好像很简单？然后我就做出了D1令我最后悔的决定，专攻“菊花图”。

  我越算到后面，越发现情况不对，这个贪心有难度啊！回头再看看时间，11点多了。T2应该来不及写了，只好孤注一掷。到后面心态真的崩了，看着时间一分一秒的流逝，我感到一阵无力，难道真的就要这样遗憾地离场了吗？

  没有反转。D1真就这么跪了。考完后遇到班里的同学，他们问我怎么样，我只能强颜欢笑。拍集体照时，大佬们都在讨论T2正解。我感觉只有我一个人T2只拿20分了。

  骑车回家的路上，我突然释然了。考差就考差了，反正我已是退役之人。无论考好考差，我不还是得退役？从高一开始就没打算在OI路上一条路走到黑。早点死心，早点投入文化课不更好？回到家后，我默默地拾起上周欠下的作业。这一切都结束了。Luogu和QQ都没有上，就怕影响心情。

Day2

  真是煎熬。昨晚没有睡好，满脑子想着CSP的事。只希望Day2不要死得像Day1那么惨。另外，我躺在床上时突然想到T2的50分做法，只是为时已晚……

  和昨天一样，考前聊天放松，上卫生间。

  Day2的解压密码倒是一遍输入正确。前二十分钟读题，更加坚定了打三题暴力的决心。

  看到T1后，只想到m=2和m=3的dp，64分。打了50分钟左右。剩下那些点我也没有去想了。

  T2，想先打12分的dfs，最后发现能拿20多分。

  T3，先打了情况A，然后再打暴力。本以为能打50分，考完后发现情况A打炸了，直接没了15分。

  Day2比Day1更难，CCF这次真的没有给我这样的菜鸡留点颜面。2019年11月17日中午12:00，监考老师宣布完考试结束，我的OI生涯也落下了帷幕。学了将近6年OI，最后只会打6题暴力，唉！

  考完后，张老师给我们几个合了影，然后我们就各自回班拿文化课的作业了。Qxx要准备周三的高三月考，我和Lj也有一堆周末作业。毕竟，我们都是业余选手，文化课才是主科，OI生活只是陪衬。现在要做的就是及时收心，投入文化课的学习。

  尽管如此，这段经历我将永远不会忘记。或许多年后，回想高中时的OI生涯，想到能有一群志同道合的人一起奋斗，想起曾经的奋斗与努力，没能取得好成绩的挫败感，也早已经被时间淡去了。毕竟，我努力过，我没有遗憾。

  THE END IS NEVER THE END.
